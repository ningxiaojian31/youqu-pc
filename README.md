# youqu-web
“友趣社区”后台管理系统

* #### 简述：

   - 使用VUE+ElementUI搭建

* #### 启动：

   1.安装node.js
   
   2.安装：
   
         npm install -g cnpm --registry=https://registry.npm.taobao.org
         
         cnpm install
         
   3.启动:
   
         npm run dev
   
   4.API请求路径：src/api/api.js


   
* #### 默认访问路径：
      http://localhost:8080
